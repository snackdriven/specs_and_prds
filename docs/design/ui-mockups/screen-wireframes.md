# UI Mockups & Wireframes

## Design Principles

**ADHD-Friendly UI:**
- Clear visual hierarchy
- Minimal cognitive load
- Quick access to common actions
- Progress visualization everywhere
- Forgiving interactions (easy undo)
- No overwhelming walls of text

**Mobile-First:**
- Thumb-friendly tap targets
- Swipe gestures for common actions
- Bottom navigation for key features
- Quick capture modes

## Design System Elements

### Color Palette

**Journal Templates:**
- Daily Reflection: Deep Blue (#2D3E50)
- Morning Intention: Warm Orange (#F39C12)
- Gratitude: Soft Gold (#F1C40F)
- Problem Solving: Bright Yellow (#F4D03F)
- Creative: Purple (#9B59B6)
- Anxiety: Calming Blue (#5DADE2)
- Work Review: Professional Navy (#34495E)
- Weekly Review: Deep Green (#27AE60)

**Chore Categories:**
- Vehicle: Black (#2C3E50)
- Home: Brown (#8D6E63)
- Health: Medical Blue (#2196F3)
- Equipment: Tech Blue (#2196F3)

**UI Elements:**
- Primary: Blue (#2196F3)
- Success: Green (#4CAF50)
- Warning: Orange (#FF9800)
- Error: Red (#E74C3C)
- Neutral: Gray (#95A5A6)

### Typography

**Headers:**
- H1: 28px, Bold
- H2: 22px, Semibold
- H3: 18px, Semibold

**Body:**
- Regular: 16px, Regular
- Small: 14px, Regular
- Tiny: 12px, Regular

### Spacing

**Consistent padding:**
- Screen edges: 16px
- Between sections: 24px
- Between items: 12px
- Within cards: 16px

### Interactions

**Touch targets:**
- Minimum: 44x44px
- Buttons: 48px height
- Icons: 24x24px (in 44px tap area)

**Feedback:**
- Buttons: Scale down 95% on press
- Lists: Subtle highlight on tap
- Swipe actions: Reveal action buttons
- Long press: Context menus

### Accessibility

**WCAG AA Compliance:**
- Color contrast: 4.5:1 minimum
- Text size: Adjustable in settings
- Screen reader: Full support
- Haptic feedback: Optional

## Responsive Breakpoints

**Mobile (default):** 320px - 480px
**Tablet:** 481px - 768px
**Desktop (future):** 769px+

**Tablet Adaptations:**
- Two-column layouts for template grids
- Side panel for navigation
- Larger text entry areas

---

## Animation Principles

**Subtle, purposeful:**
- Screen transitions: Slide (300ms)
- Modal appearance: Fade + scale (200ms)
- Checkbox toggle: Bounce (150ms)
- Progress bars: Smooth fill (400ms)
- Never distract from content
