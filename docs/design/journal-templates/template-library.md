# Journal Template Library

## Design Principles

**ADHD-Friendly Design:**
- Short, focused prompts (3-7 questions max)
- Visual progress indicators
- Optional sections for flexibility
- Quick-capture modes for low-energy days
- Clear time estimates upfront

**Integration Features:**
- Mood snapshot capture (before/after/both)
- Spotify "now playing" auto-capture
- Rich text with simple formatting
- Tags for easy retrieval
- Streak tracking for motivation

---

**Purpose:** End-of-day review and planning
**Category:** Daily
**Estimated Time:** 5-7 minutes
**Icon:** üåô | **Color:** Deep Blue (#2D3E50)

**When to Use:** Every evening before bed

**Sections:**

**Mood Prompt:** After (captures end-of-day emotional state)
**Spotify Capture:** Auto (saves what you're listening to while journaling)
**Suggested Frequency:** Daily, 8-10 PM

## Template Customization System

**User Can Modify:**
- Edit prompts/questions
- Add/remove sections
- Change required/optional status
- Adjust time estimates
- Change mood/Spotify settings
- Create entirely custom templates

**Template Versioning:**
- Save custom versions as new templates
- Reset to default anytime
- Share templates between devices

**Smart Features:**
- Suggested templates based on time of day
- Streak tracking per template
- Completion analytics
- Favorite templates quick access
