# Git Configuration Notes

## Default Branch Name
**Preference:** Use `main` as the default branch name (not `master`)

**When creating new repositories:**
- Initialize with: `git init -b main` or `git branch -M main`
- Push to: `git push -u origin main`
- Always use `main` as the default branch name
- When using `gh repo create`, it will use `main` by default

**Remember:** User prefers `main` over `master` for all new repositories.

**Current Status:**
- ✅ Both livejournal-clone and jira-wrapper now use `main` as default branch
- ✅ Local branches renamed to `main`
- ✅ Remote default branches set to `main` on GitHub
